# UGJB ప్లాట్‌ఫారమ్

[![GitHub](https://img.shields.io/badge/github-repo-blue?logo=github)](https://github.com/qtvhao/UGJB)
[![License](https://img.shields.io/github/license/qtvhao/UGJB.svg?color=blue)](LICENSE)

**ఏకీకృత వర్క్‌ఫోర్స్ మరియు ఇంజినీరింగ్ అనలిటిక్స్ ప్లాట్‌ఫారమ్** - HR నిర్వహణ మరియు ఇంజినీరింగ్ పనితీరు విశ్లేషణను ఒకే ఓపెన్-సోర్స్ సిస్టమ్‌లో ఏకీకృతం చేసే మాడ్యులర్ ప్లాట్‌ఫారమ్.

## అవలోకనం

UGJB ప్లాట్‌ఫారమ్ రియల్-టైమ్ అంతర్దృష్టుల ద్వారా ప్రతిభా నిర్ణయాలను సాంకేతిక ఫలితాలతో సమలేఖనం చేయడానికి సంస్థలను అనుమతిస్తుంది. ఇది విచ్ఛిన్నమైన SaaS పరిష్కారాలను తొలగిస్తుంది, ప్రామాణిక ఏకీకరణ నమూనాలు మరియు పునర్వినియోగ భాగాలతో ఎంటర్‌ప్రైజ్-గ్రేడ్ సామర్థ్యాలను కలుపుతుంది, అదే సమయంలో మొత్తం యాజమాన్య వ్యయాన్ని తగ్గిస్తుంది.

### ముఖ్య లక్షణాలు

- **ఏకీకృత వర్క్‌ఫోర్స్ నిర్వహణ** - ఉద్యోగి ప్రొఫైల్స్, నైపుణ్యాల ట్రాకింగ్, FTE కేటాయింపు మరియు పని స్థితి
- **ఇంజినీరింగ్ అనలిటిక్స్** - DORA మెట్రిక్స్, కోడ్ నాణ్యత స్కోర్లు మరియు విశ్వసనీయత సూచికలు
- **డెవలప్‌మెంట్ టూల్స్ ఏకీకరణ** - Jira, GitLab, Firebase Crashlytics, Prometheus
- **రియల్-టైమ్ డాష్‌బోర్డ్స్** - అనుకూలీకరించదగిన KPI విజువలైజేషన్స్ మరియు నివేదికలు
- **రోల్-ఆధారిత యాక్సెస్ కంట్రోల్** - సూక్ష్మ అనుమతులు మరియు డేటా భద్రత
- **ఓపెన్-సోర్స్ మరియు మాడ్యులర్** - విస్తరించదగిన నిర్మాణం, విక్రేత లాక్-ఇన్ లేదు

## త్వరిత ప్రారంభం

### ముందస్తు అవసరాలు

- Docker మరియు Docker Compose
- Git

### ఇన్‌స్టాలేషన్

```bash
# రిపోజిటరీని క్లోన్ చేయండి
git clone https://github.com/qtvhao/UGJB.git
cd UGJB

# అన్ని సేవలను ప్రారంభించండి
docker-compose up -d

# హెల్త్ ఎండ్‌పాయింట్‌ను ధృవీకరించండి
curl http://localhost:8080/health
```

### ప్లాట్‌ఫారమ్ యాక్సెస్

- **Web UI**: http://localhost:8081
- **API Gateway**: http://localhost:8080

## నిర్మాణం

UGJB స్పష్టంగా నిర్వచించబడిన బౌండెడ్ కాంటెక్స్ట్‌లతో మైక్రోసర్వీసెస్-ఆధారిత నిర్మాణాన్ని అనుసరిస్తుంది:

- **HR నిర్వహణ** - ఉద్యోగి రిజిస్ట్రీ మరియు కేటాయింపు ఇంజిన్
- **ఇంజినీరింగ్ అనలిటిక్స్** - మెట్రిక్స్ కలెక్టర్, KPI ఇంజిన్, అంతర్దృష్టుల డాష్‌బోర్డ్
- **లక్ష్య నిర్వహణ** - ఆబ్జెక్టివ్స్ మరియు కీ రిజల్ట్స్ ట్రాకింగ్
- **ప్రాజెక్ట్ నిర్వహణ** - స్ప్రింట్ సమన్వయం మరియు టాస్క్ అసైన్‌మెంట్
- **సిస్టమ్ ఏకీకరణ** - డేటా పైప్‌లైన్ మరియు API గేట్‌వే
- **వర్క్‌ఫోర్స్ వెల్‌బీయింగ్** - బర్నౌట్ అంచనా మరియు శ్రేయస్సు పర్యవేక్షణ

## UGJB ఎందుకు?

### పరిష్కరించబడిన సమస్యలు

1. **ఏకీకరణ విభజన** - Firebase, Prometheus, GitLab మరియు Jira నుండి డేటాను ఏకీకృతం చేస్తుంది
2. **డొమైన్ సైలోలు** - HR నైపుణ్యాల నిర్వహణను ఇంజినీరింగ్ KPI లతో కనెక్ట్ చేస్తుంది
3. **ఖర్చు అడ్డంకులు** - ≤ $120k 3 సంవత్సరాల TCO vs $200k+ ఎంటర్‌ప్రైజ్ SaaS
4. **అనుకూలీకరణ పరిమితులు** - ప్లాట్‌ఫారమ్ స్థిరత్వాన్ని నిర్వహించే విస్తరించదగిన వర్క్‌ఫ్లోలు

### విజయ మెట్రిక్స్

| మెట్రిక్ | బేస్‌లైన్ | లక్ష్యం |
|----------|-----------|---------|
| 3 సంవత్సరాల మొత్తం యాజమాన్య వ్యయం | $201k-$246k | ≤ $120k |
| ఏకీకరణ కవరేజ్ | 50% GitLab | 100% కవరేజ్ |
| అంతర్దృష్టుల సమయం | 72+ గంటలు | ≤ 2 గంటలు |
| ప్లాట్‌ఫారమ్ అప్‌టైమ్ | నిర్వచించబడలేదు | ≥ 99.9% |

## ప్రాథమిక ఉపయోగం

### ఉద్యోగుల నిర్వహణ

```bash
# API ద్వారా ఉద్యోగి ప్రొఫైల్‌ను సృష్టించండి
curl -X POST http://localhost:8080/api/v1/employees \
  -H "Content-Type: application/json" \
  -d '{
    "name": "రమేష్ కుమార్",
    "role": "సీనియర్ డెవలపర్",
    "department": "ఇంజినీరింగ్",
    "status": "active",
    "fte": 100
  }'
```

### ఇంజినీరింగ్ మెట్రిక్స్ చూడండి

ఇంజినీరింగ్ మెట్రిక్స్ డాష్‌బోర్డ్‌ను యాక్సెస్ చేయండి:
- DORA మెట్రిక్స్ (డిప్లాయ్‌మెంట్ ఫ్రీక్వెన్సీ, లీడ్ టైమ్)
- కోడ్ నాణ్యత స్కోర్లు
- ఇటీవలి డిప్లాయ్‌మెంట్లు
- టీమ్ ఇంజినీరింగ్ అవుట్‌పుట్

### ఏకీకరణలను కాన్ఫిగర్ చేయండి

Web UI ద్వారా బాహ్య సాధనాలను కనెక్ట్ చేయండి:
1. "ఏకీకరణలు"కు నావిగేట్ చేయండి
2. సాధన రకాన్ని ఎంచుకోండి (Jira, GitLab, Firebase, Prometheus)
3. API ఎండ్‌పాయింట్ మరియు ప్రమాణీకరణను నమోదు చేయండి
4. సింక్ ఫ్రీక్వెన్సీని సెట్ చేయండి
