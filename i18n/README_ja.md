# UGJB プラットフォーム

[![GitHub](https://img.shields.io/badge/github-repo-blue?logo=github)](https://github.com/qtvhao/UGJB)
[![License](https://img.shields.io/github/license/qtvhao/UGJB.svg?color=blue)](LICENSE)

**統合ワークフォースおよびエンジニアリング分析プラットフォーム** - HR管理とエンジニアリングパフォーマンス分析を1つのオープンソースシステムに統合するモジュラープラットフォーム。

## 概要

UGJBプラットフォームは、リアルタイムの洞察を通じて、組織が人材決定を技術的成果と整合させることを可能にします。分散したSaaSソリューションを排除し、標準化された統合パターンと再利用可能なコンポーネントとともにエンタープライズグレードの機能を組み合わせ、総所有コストを削減します。

### 主な機能

- **統合ワークフォース管理** - 従業員プロファイル、スキル追跡、FTE割り当て、勤務状況
- **エンジニアリング分析** - DORAメトリクス、コード品質スコア、信頼性指標
- **開発ツールとの統合** - Jira、GitLab、Firebase Crashlytics、Prometheus
- **リアルタイムダッシュボード** - カスタマイズ可能なKPI可視化とレポート
- **ロールベースのアクセス制御** - きめ細かい権限とデータセキュリティ
- **オープンソースでモジュラー** - 拡張可能なアーキテクチャ、ベンダーロックインなし

## クイックスタート

### 前提条件

- Docker と Docker Compose
- Git

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/qtvhao/UGJB.git
cd UGJB

# すべてのサービスを起動
docker-compose up -d

# ヘルスエンドポイントを確認
curl http://localhost:8080/health
```

### プラットフォームへのアクセス

- **Web UI**: http://localhost:8081
- **API Gateway**: http://localhost:8080

## アーキテクチャ

UGJBは、明確に定義された境界コンテキストを持つマイクロサービスベースのアーキテクチャに従います：

- **HR管理** - 従業員レジストリと割り当てエンジン
- **エンジニアリング分析** - メトリクスコレクター、KPIエンジン、インサイトダッシュボード
- **目標管理** - 目標と主要結果の追跡
- **プロジェクト管理** - スプリント調整とタスク割り当て
- **システム統合** - データパイプラインとAPIゲートウェイ
- **ワークフォースウェルビーイング** - バーンアウト予測とウェルビーイング監視

## なぜUGJB？

### 解決される問題

1. **統合の断片化** - Firebase、Prometheus、GitLab、Jiraからのデータを統合
2. **ドメインサイロ** - HRスキル管理をエンジニアリングKPIと接続
3. **コストの障壁** - ≤ $120k 3年間TCO vs $200k+ エンタープライズSaaS
4. **カスタマイズの制限** - プラットフォームの安定性を維持する拡張可能なワークフロー

### 成功指標

| 指標 | ベースライン | 目標 |
|------|-------------|------|
| 3年間総所有コスト | $201k-$246k | ≤ $120k |
| 統合カバレッジ | 50% GitLab | 100% カバレッジ |
| インサイトまでの時間 | 72+ 時間 | ≤ 2 時間 |
| プラットフォーム稼働時間 | 未定義 | ≥ 99.9% |

## 基本的な使用方法

### 従業員管理

```bash
# API経由で従業員プロファイルを作成
curl -X POST http://localhost:8080/api/v1/employees \
  -H "Content-Type: application/json" \
  -d '{
    "name": "山田太郎",
    "role": "シニア開発者",
    "department": "エンジニアリング",
    "status": "active",
    "fte": 100
  }'
```

### エンジニアリングメトリクスの表示

エンジニアリングメトリクスダッシュボードにアクセス：
- DORAメトリクス（デプロイ頻度、リードタイム）
- コード品質スコア
- 最近のデプロイ
- チームのエンジニアリング成果物

### 統合の設定

Web UIを介して外部ツールに接続：
1. 「統合」に移動
2. ツールタイプを選択（Jira、GitLab、Firebase、Prometheus）
3. APIエンドポイントと認証を入力
4. 同期頻度を設定
