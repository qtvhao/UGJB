# UGJB প্ল্যাটফর্ম

[![GitHub](https://img.shields.io/badge/github-repo-blue?logo=github)](https://github.com/qtvhao/UGJB)
[![License](https://img.shields.io/github/license/qtvhao/UGJB.svg?color=blue)](LICENSE)

**সমন্বিত কর্মশক্তি এবং প্রকৌশল বিশ্লেষণ প্ল্যাটফর্ম** - একটি মডুলার প্ল্যাটফর্ম যা এইচআর ম্যানেজমেন্ট এবং ইঞ্জিনিয়ারিং পারফরম্যান্স বিশ্লেষণকে একটি একক ওপেন-সোর্স সিস্টেমে একীভূত করে।

## সংক্ষিপ্ত বিবরণ

UGJB প্ল্যাটফর্ম সংস্থাগুলিকে রিয়েল-টাইম অন্তর্দৃষ্টির মাধ্যমে প্রতিভা সিদ্ধান্তগুলিকে প্রযুক্তিগত ফলাফলের সাথে সারিবদ্ধ করতে সক্ষম করে। এটি খণ্ডিত SaaS সমাধানগুলি দূর করে এন্টারপ্রাইজ-গ্রেড ক্ষমতাগুলিকে মানসম্মত ইন্টিগ্রেশন প্যাটার্ন এবং পুনর্ব্যবহারযোগ্য উপাদানগুলির সাথে একত্রিত করে, যখন মোট মালিকানা খরচ হ্রাস করে।

### মূল বৈশিষ্ট্যসমূহ

- **সমন্বিত কর্মশক্তি ম্যানেজমেন্ট** - কর্মচারী প্রোফাইল, দক্ষতা ট্র্যাকিং, FTE বরাদ্দকরণ এবং কাজের অবস্থা
- **ইঞ্জিনিয়ারিং বিশ্লেষণ** - DORA মেট্রিক্স, কোড গুণমান স্কোর এবং নির্ভরযোগ্যতা সূচক
- **ডেভেলপমেন্ট টুলস ইন্টিগ্রেশন** - Jira, GitLab, Firebase Crashlytics, Prometheus
- **রিয়েল-টাইম ড্যাশবোর্ড** - কাস্টমাইজযোগ্য KPI ভিজুয়ালাইজেশন এবং রিপোর্ট
- **ভূমিকা-ভিত্তিক অ্যাক্সেস নিয়ন্ত্রণ** - সূক্ষ্ম অনুমতি এবং ডেটা সুরক্ষা
- **ওপেন-সোর্স এবং মডুলার** - সম্প্রসারণযোগ্য আর্কিটেকচার, কোনো ভেন্ডর লক-ইন নেই

## দ্রুত শুরু

### পূর্বশর্ত

- Docker এবং Docker Compose
- Git

### ইনস্টলেশন

```bash
# রিপোজিটরি ক্লোন করুন
git clone https://github.com/qtvhao/UGJB.git
cd UGJB

# সমস্ত সেবা শুরু করুন
docker-compose up -d

# স্বাস্থ্য এন্ডপয়েন্ট যাচাই করুন
curl http://localhost:8080/health
```

### প্ল্যাটফর্ম অ্যাক্সেস

- **Web UI**: http://localhost:8081
- **API Gateway**: http://localhost:8080

## স্থাপত্য

UGJB স্পষ্টভাবে সংজ্ঞায়িত সীমাবদ্ধ প্রসঙ্গ সহ মাইক্রোসার্ভিস-ভিত্তিক আর্কিটেকচার অনুসরণ করে:

- **এইচআর ম্যানেজমেন্ট** - কর্মচারী রেজিস্ট্রি এবং বরাদ্দকরণ ইঞ্জিন
- **ইঞ্জিনিয়ারিং বিশ্লেষণ** - মেট্রিক্স সংগ্রাহক, KPI ইঞ্জিন, অন্তর্দৃষ্টি ড্যাশবোর্ড
- **লক্ষ্য ম্যানেজমেন্ট** - উদ্দেশ্য এবং মূল ফলাফল ট্র্যাকিং
- **প্রকল্প ম্যানেজমেন্ট** - স্প্রিন্ট সমন্বয় এবং কাজ নিয়োগ
- **সিস্টেম ইন্টিগ্রেশন** - ডেটা পাইপলাইন এবং API গেটওয়ে
- **কর্মশক্তি সুস্থতা** - বার্নআউট পূর্বাভাস এবং সুস্থতা পর্যবেক্ষণ

## কেন UGJB?

### সমাধান করা সমস্যা

1. **ইন্টিগ্রেশন খণ্ডিতকরণ** - Firebase, Prometheus, GitLab এবং Jira থেকে ডেটা একীভূত করে
2. **ডোমেইন সাইলো** - এইচআর দক্ষতা ম্যানেজমেন্টকে ইঞ্জিনিয়ারিং KPIs এর সাথে সংযুক্ত করে
3. **খরচ বাধা** - ≤ $120k 3 বছর TCO বনাম $200k+ এন্টারপ্রাইজ SaaS
4. **কাস্টমাইজেশন সীমাবদ্ধতা** - সম্প্রসারণযোগ্য ওয়ার্কফ্লো যা প্ল্যাটফর্ম স্থিতিশীলতা বজায় রাখে

### সাফল্যের মেট্রিক্স

| মেট্রিক | বেসলাইন | লক্ষ্য |
|---------|---------|------|
| 3 বছরের মোট মালিকানা খরচ | $201k-$246k | ≤ $120k |
| ইন্টিগ্রেশন কভারেজ | 50% GitLab | 100% কভারেজ |
| অন্তর্দৃষ্টির সময় | 72+ ঘণ্টা | ≤ 2 ঘণ্টা |
| প্ল্যাটফর্ম আপটাইম | অনির্ধারিত | ≥ 99.9% |

## মৌলিক ব্যবহার

### কর্মচারী পরিচালনা

```bash
# API এর মাধ্যমে কর্মচারী প্রোফাইল তৈরি করুন
curl -X POST http://localhost:8080/api/v1/employees \
  -H "Content-Type: application/json" \
  -d '{
    "name": "রহিম আহমেদ",
    "role": "সিনিয়র ডেভেলপার",
    "department": "ইঞ্জিনিয়ারিং",
    "status": "active",
    "fte": 100
  }'
```

### ইঞ্জিনিয়ারিং মেট্রিক্স দেখুন

ইঞ্জিনিয়ারিং মেট্রিক্স ড্যাশবোর্ড অ্যাক্সেস করুন:
- DORA মেট্রিক্স (স্থাপনার ফ্রিকোয়েন্সি, লিড টাইম)
- কোড গুণমান স্কোর
- সাম্প্রতিক স্থাপনা
- টিম ইঞ্জিনিয়ারিং আউটপুট

### ইন্টিগ্রেশন কনফিগার করুন

Web UI এর মাধ্যমে বাহ্যিক টুল সংযোগ করুন:
1. "ইন্টিগ্রেশন" এ নেভিগেট করুন
2. টুল টাইপ নির্বাচন করুন (Jira, GitLab, Firebase, Prometheus)
3. API এন্ডপয়েন্ট এবং প্রমাণীকরণ লিখুন
4. সিঙ্ক ফ্রিকোয়েন্সি সেট করুন
