# UGJB پلیٹ فارم

[![GitHub](https://img.shields.io/badge/github-repo-blue?logo=github)](https://github.com/qtvhao/UGJB)
[![License](https://img.shields.io/github/license/qtvhao/UGJB.svg?color=blue)](LICENSE)

**یکجا ورک فورس اور انجینئرنگ تجزیات پلیٹ فارم** - ایک ماڈیولر پلیٹ فارم جو HR مینجمنٹ اور انجینئرنگ کارکردگی کے تجزیے کو ایک اوپن سورس سسٹم میں ضم کرتا ہے۔

## جائزہ

UGJB پلیٹ فارم تنظیموں کو ریئل ٹائم بصیرت کے ذریعے ٹیلنٹ کے فیصلوں کو تکنیکی نتائج کے ساتھ ہم آہنگ کرنے کی اجازت دیتا ہے۔ یہ منتشر SaaS حل کو ختم کرتا ہے، معیاری انضمام کے نمونوں اور دوبارہ قابل استعمال اجزاء کے ساتھ انٹرپرائز گریڈ صلاحیتوں کو یکجا کرتا ہے، جبکہ کل ملکیت کی لاگت کو کم کرتا ہے۔

### اہم خصوصیات

- **یکجا ورک فورس مینجمنٹ** - ملازمین کی پروفائلز، مہارتوں کی ٹریکنگ، FTE مختص کرنا اور کام کی حیثیت
- **انجینئرنگ تجزیات** - DORA میٹرکس، کوڈ کوالٹی سکور اور وشوسنییتا کے اشارے
- **ترقیاتی ٹولز کے ساتھ انضمام** - Jira، GitLab، Firebase Crashlytics، Prometheus
- **ریئل ٹائم ڈیش بورڈز** - حسب ضرورت KPI تصویری نمائش اور رپورٹس
- **رول پر مبنی رسائی کنٹرول** - تفصیلی اجازتیں اور ڈیٹا سیکیورٹی
- **اوپن سورس اور ماڈیولر** - توسیعی فن تعمیر، کوئی وینڈر لاک ان نہیں

## فوری آغاز

### ضروریات

- Docker اور Docker Compose
- Git

### تنصیب

```bash
# ریپوزٹری کلون کریں
git clone https://github.com/qtvhao/UGJB.git
cd UGJB

# تمام سروسز شروع کریں
docker-compose up -d

# صحت کا اختتامی نقطہ تصدیق کریں
curl http://localhost:8080/health
```

### پلیٹ فارم تک رسائی

- **Web UI**: http://localhost:8081
- **API Gateway**: http://localhost:8080

## فن تعمیر

UGJB واضح طور پر متعین محدود سیاق و سباق کے ساتھ مائیکرو سروسز پر مبنی فن تعمیر کی پیروی کرتا ہے:

- **HR مینجمنٹ** - ملازمین کی رجسٹری اور مختص کرنے کا انجن
- **انجینئرنگ تجزیات** - میٹرکس جمع کرنے والا، KPI انجن، بصیرت ڈیش بورڈ
- **مقصد کی مینجمنٹ** - مقاصد اور کلیدی نتائج کی ٹریکنگ
- **پروجیکٹ مینجمنٹ** - سپرنٹ کوآرڈینیشن اور ٹاسک تفویض
- **سسٹم انضمام** - ڈیٹا پائپ لائن اور API گیٹ وے
- **ورک فورس ویل بیئنگ** - برن آؤٹ کی پیش گوئی اور فلاح و بہبود کی نگرانی

## UGJB کیوں؟

### حل شدہ مسائل

1. **انضمام کی تقسیم** - Firebase، Prometheus، GitLab اور Jira سے ڈیٹا کو متحد کرتا ہے
2. **ڈومین سائلوز** - HR مہارت کی مینجمنٹ کو انجینئرنگ KPIs سے جوڑتا ہے
3. **لاگت کی رکاوٹیں** - ≤ $120k 3 سال TCO بمقابلہ $200k+ انٹرپرائز SaaS
4. **حسب ضرورت کی حدود** - توسیعی ورک فلو جو پلیٹ فارم کی استحکام برقرار رکھتے ہیں

### کامیابی کے میٹرکس

| میٹرک | بیس لائن | ہدف |
|--------|----------|------|
| 3 سال کل ملکیت کی لاگت | $201k-$246k | ≤ $120k |
| انضمام کوریج | 50% GitLab | 100% کوریج |
| بصیرت تک وقت | 72+ گھنٹے | ≤ 2 گھنٹے |
| پلیٹ فارم اپ ٹائم | غیر متعین | ≥ 99.9% |

## بنیادی استعمال

### ملازمین کا انتظام

```bash
# API کے ذریعے ملازم کی پروفائل بنائیں
curl -X POST http://localhost:8080/api/v1/employees \
  -H "Content-Type: application/json" \
  -d '{
    "name": "احمد علی",
    "role": "سینئر ڈیولپر",
    "department": "انجینئرنگ",
    "status": "active",
    "fte": 100
  }'
```

### انجینئرنگ میٹرکس دیکھیں

انجینئرنگ میٹرکس ڈیش بورڈ تک رسائی:
- DORA میٹرکس (تعیناتی کی تعدد، لیڈ ٹائم)
- کوڈ کوالٹی سکور
- حالیہ تعیناتیں
- ٹیم انجینئرنگ آؤٹ پٹ

### انضمام کو ترتیب دیں

Web UI کے ذریعے بیرونی ٹولز کو منسلک کریں:
1. "انضمام" پر جائیں
2. ٹول کی قسم منتخب کریں (Jira، GitLab، Firebase، Prometheus)
3. API اختتامی نقطہ اور تصدیق درج کریں
4. مطابقت پذیری کی تعدد سیٹ کریں
