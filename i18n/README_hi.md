# UGJB प्लेटफ़ॉर्म

[![GitHub](https://img.shields.io/badge/github-repo-blue?logo=github)](https://github.com/qtvhao/UGJB)
[![License](https://img.shields.io/github/license/qtvhao/UGJB.svg?color=blue)](LICENSE)

**एकीकृत कार्यबल और इंजीनियरिंग विश्लेषण प्लेटफ़ॉर्म** - एक मॉड्यूलर प्लेटफ़ॉर्म जो HR प्रबंधन और इंजीनियरिंग प्रदर्शन विश्लेषण को एक ओपन-सोर्स सिस्टम में एकीकृत करता है।

## अवलोकन

UGJB प्लेटफ़ॉर्म संगठनों को रीयल-टाइम अंतर्दृष्टि के माध्यम से प्रतिभा निर्णयों को तकनीकी परिणामों के साथ संरेखित करने में सक्षम बनाता है। यह विखंडित SaaS समाधानों को समाप्त करता है, मानकीकृत एकीकरण पैटर्न और पुन: प्रयोज्य घटकों के साथ एंटरप्राइज-ग्रेड क्षमताओं को जोड़ता है, जबकि कुल स्वामित्व लागत को कम करता है।

### मुख्य विशेषताएं

- **एकीकृत कार्यबल प्रबंधन** - कर्मचारी प्रोफ़ाइल, कौशल ट्रैकिंग, FTE आवंटन और कार्य स्थिति
- **इंजीनियरिंग विश्लेषण** - DORA मेट्रिक्स, कोड गुणवत्ता स्कोर और विश्वसनीयता संकेतक
- **डेवलपमेंट टूल्स के साथ एकीकरण** - Jira, GitLab, Firebase Crashlytics, Prometheus
- **रीयल-टाइम डैशबोर्ड** - अनुकूलन योग्य KPI विज़ुअलाइज़ेशन और रिपोर्ट
- **भूमिका-आधारित एक्सेस नियंत्रण** - सूक्ष्म अनुमतियाँ और डेटा सुरक्षा
- **ओपन-सोर्स और मॉड्यूलर** - विस्तार योग्य आर्किटेक्चर, कोई वेंडर लॉक-इन नहीं

## त्वरित प्रारंभ

### पूर्वापेक्षाएँ

- Docker और Docker Compose
- Git

### स्थापना

```bash
# रिपॉजिटरी क्लोन करें
git clone https://github.com/qtvhao/UGJB.git
cd UGJB

# सभी सेवाओं को प्रारंभ करें
docker-compose up -d

# हेल्थ एंडपॉइंट सत्यापित करें
curl http://localhost:8080/health
```

### प्लेटफ़ॉर्म तक पहुंच

- **Web UI**: http://localhost:8081
- **API Gateway**: http://localhost:8080

## आर्किटेक्चर

UGJB स्पष्ट रूप से परिभाषित बाउंडेड कॉन्टेक्स्ट के साथ माइक्रोसर्विस-आधारित आर्किटेक्चर का अनुसरण करता है:

- **HR प्रबंधन** - कर्मचारी रजिस्ट्री और आवंटन इंजन
- **इंजीनियरिंग विश्लेषण** - मेट्रिक्स कलेक्टर, KPI इंजन, अंतर्दृष्टि डैशबोर्ड
- **लक्ष्य प्रबंधन** - उद्देश्यों और मुख्य परिणामों की ट्रैकिंग
- **परियोजना प्रबंधन** - स्प्रिंट समन्वय और कार्य असाइनमेंट
- **सिस्टम एकीकरण** - डेटा पाइपलाइन और API गेटवे
- **कार्यबल कल्याण** - बर्नआउट भविष्यवाणी और कल्याण निगरानी

## UGJB क्यों?

### हल की गई समस्याएं

1. **एकीकरण विखंडन** - Firebase, Prometheus, GitLab और Jira से डेटा एकीकृत करता है
2. **डोमेन साइलो** - HR कौशल प्रबंधन को इंजीनियरिंग KPI से जोड़ता है
3. **लागत बाधाएं** - ≤ $120k 3 वर्ष TCO बनाम $200k+ एंटरप्राइज़ SaaS
4. **अनुकूलन सीमाएं** - प्लेटफ़ॉर्म स्थिरता बनाए रखने वाले विस्तार योग्य वर्कफ़्लो

### सफलता मेट्रिक्स

| मेट्रिक | आधार रेखा | लक्ष्य |
|---------|-----------|--------|
| 3 वर्ष कुल स्वामित्व लागत | $201k-$246k | ≤ $120k |
| एकीकरण कवरेज | 50% GitLab | 100% कवरेज |
| अंतर्दृष्टि का समय | 72+ घंटे | ≤ 2 घंटे |
| प्लेटफ़ॉर्म अपटाइम | अपरिभाषित | ≥ 99.9% |

## बुनियादी उपयोग

### कर्मचारियों का प्रबंधन

```bash
# API के माध्यम से कर्मचारी प्रोफ़ाइल बनाएं
curl -X POST http://localhost:8080/api/v1/employees \
  -H "Content-Type: application/json" \
  -d '{
    "name": "राहुल शर्मा",
    "role": "सीनियर डेवलपर",
    "department": "इंजीनियरिंग",
    "status": "active",
    "fte": 100
  }'
```

### इंजीनियरिंग मेट्रिक्स देखें

इंजीनियरिंग मेट्रिक्स डैशबोर्ड तक पहुंचें:
- DORA मेट्रिक्स (तैनाती आवृत्ति, लीड टाइम)
- कोड गुणवत्ता स्कोर
- हालिया तैनाती
- टीम इंजीनियरिंग आउटपुट

### एकीकरण कॉन्फ़िगर करें

Web UI के माध्यम से बाहरी टूल कनेक्ट करें:
1. "एकीकरण" पर नेविगेट करें
2. टूल प्रकार चुनें (Jira, GitLab, Firebase, Prometheus)
3. API एंडपॉइंट और प्रमाणीकरण दर्ज करें
4. सिंक आवृत्ति सेट करें
